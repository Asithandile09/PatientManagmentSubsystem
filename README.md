# Patient Management System

A hospital dashboard and patient management system built with ASP.NET Core MVC, Entity Framework, and Bootstrap.

## Features
- Patient admission and folder management
- Bed grid view and discharged patients list
- CRUD operations for patients and staff
- Dashboard styled with Bootstrap for responsive design

## Tech Stack
- ASP.NET Core MVC
- Entity Framework Core
- Bootstrap 5
- SQL Server / LocalDB

## How to Run
1. Open the project in Visual Studio.
2. Restore NuGet packages.
3. Build the project.
4. Run using IIS Express or Kestrel server.
5. Use the dashboard to add/view patients and manage folders.
